# Specialists

- `frontend-agent.md`: user interface, browser behavior, accessibility, and responsive design.
- `backend-agent.md`: APIs, persistence, jobs, integrations, and server behavior.
- `security-agent.md`: trust boundaries, authentication, authorization, and sensitive data.
- `qa-agent.md`: automated, browser, accessibility, visual, and release validation.

Use the narrowest applicable specialist. Cross-cutting work is coordinated through `agents/orchestrator/workflow.md`.
