# Generated And Sensitive Files

Do not hand-edit:

- `node_modules/`, `.pnpm-store/`, package-manager caches;
- `dist/`, `build/`, `.next/`, `.nuxt/`, `.svelte-kit/`, `.angular/`, `.vite/`, cache folders;
- coverage, test reports, screenshots, traces, and videos unless explicitly retained as evidence;
- generated API clients, ORM clients, route manifests, and schema output;
- minified bundles and source maps;
- `.env*`, credentials, private keys, certificates, production exports, or logs containing sensitive data.

Lockfiles are source-controlled but tool-generated. Change them only through the approved package manager and review dependency deltas.

Approved generated artifacts must record their generator command, source input, and verification.
