# Web Task Router

| Task | Primary | Required Supporting Review |
|---|---|---|
| Component, page, layout, styling | Frontend | QA |
| Form or client-side mutation | Frontend | Backend or Security when data crosses trust boundary |
| API, server route, job, webhook | Backend | Security and QA |
| Database schema or migration | Backend | Security and QA |
| Authentication or authorization | Security | Backend and QA |
| Accessibility or responsive bug | Frontend | QA |
| Performance | Owning layer | QA with measured baseline |
| Dependency, build, CI, deployment | Orchestrator | Owning specialist and QA |
| Architecture or broad feature | Orchestrator | Frontend, Backend, Security, and QA as applicable |
